# Support Guide

We offer several channels for developers integrating with the Sera Protocol SDK.

---

## 1. GitHub Discussions

For integration questions, developer support, and troubleshooting:
👉 [Visit GitHub Discussions](https://github.com/sera-protocol/sdk/discussions)

Please check the FAQ and search discussions before creating a new post.

---

## 2. Discord Community

For real-time discussions, announcements, and chat with other contributors:
👉 [Join Discord](https://discord.gg/seraprotocol)

---

## 3. Bug Reports

If you've identified a bug inside the SDK, please submit an issue using our bug report template:
👉 [Open a Bug Issue](https://github.com/sera-protocol/sdk/issues/new)

For security concerns, please refer directly to our [SECURITY.md](./SECURITY.md) guidelines.
